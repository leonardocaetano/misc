use the default pacman mirror
if your efi partition is monted on /boot instead of /boot/efi, when you point to it on grub-install just point to it, no /eft subfolder will be created
don't install anything boot releated other then grub
generally, this is your partation table:

/dev/sda1 - efi partition 512M fat32
/dev/sda2 - ext4 partition, all the rest of your storage device, but the 512M for efi and 2GB for swap
/dev/sda3 - swap partition, i have plenty of ram, but this is just in case

don't forget to generate the fstab using GUID (-U command!!)

packages to install while you are in chroot

networkmanager (don't forget to activate its initalization on systemd!!!!!)
vim and nvim
grub
sudo
efibootmgr

then you create your leo user, with the wheel group (which is enough to have sudo access)
